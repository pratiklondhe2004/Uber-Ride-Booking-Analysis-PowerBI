# Uber-Ride-Booking-Analysis-PowerBI

## Project Overview

This Power BI dashboard analyzes Uber ride booking data to monitor business performance, booking trends, revenue generation, customer behavior, and vehicle utilization.
The dashboard provides interactive visualizations and KPIs that help stakeholders understand ride demand, profitability, cancellations, customer satisfaction, and operational efficiency.

## Objectives
- Analyze ride booking performance.
- Track revenue and profit trends.
- Identify top pickup and drop locations.
- Compare vehicle type performance.
- Monitor customer and driver ratings.
- Evaluate booking completion and cancellation rates.
